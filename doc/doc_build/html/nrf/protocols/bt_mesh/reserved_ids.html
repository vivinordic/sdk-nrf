Reserved vendor model IDs and opcodes

  • Reserved vendor model IDs

  • Reserved opcodes

This page contains information about vendor model IDs and opcodes allocated by Nordic Semiconductor.

As a Bluetooth® Special Interest Group (SIG) member, Nordic Semiconductor has been assigned the Company ID 0x0059. You can find this information on the Bluetooth SIG company identifiers page.
